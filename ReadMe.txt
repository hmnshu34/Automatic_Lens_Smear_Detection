
The Source code is a single file named detect_smear.py

The results can be seen in FinalResult.jpg 

And the masked image required is mask_img.jpg

Packages required to run the code:
	1. Python 2.7
	2. Opencv 3.1.0
	3. Numpy
	4. Scipy

Command line to run the code(The default mode is debug):
	python detect_smear.py -d "C:\Users\suraj\Downloads\sample_drive\sample_drive\cam_2"

If you don't want to see the ouput of every stage please run it in the release mode
	python detect_smear.py -d "C:\Users\suraj\Downloads\sample_drive\sample_drive\cam_2" -r True
